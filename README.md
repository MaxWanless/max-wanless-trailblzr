## Tech Stack

### Front-End

-React
-SASS
-Axios
-JWT-Decode
-Material UI

### Back-End

- Node.js
- Express
- MySQL
- knex.js
- JSON Web Token
- bcrypt
- REST API

## Installation Instructions

- `npm run i` in root folder.
- `npm run dev` in root folder.
- If port is already in use, use command `killall -9 node` and start again.
